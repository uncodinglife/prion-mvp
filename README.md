\# Proyecto Prion - MVP



Juego PWA de supervivencia zombie geolocalizado.



Encuentros físicos reales entre jugadores en Sant Feliu de Guíxols, resueltos con sistema de dados sobre una mecánica asimétrica civiles vs zombies.



Stack técnico:

\- SvelteKit (frontend)

\- Supabase (backend, PostgreSQL + PostGIS)

\- Leaflet (mapas)

\- Vercel (despliegue)



Estado: en desarrollo.
